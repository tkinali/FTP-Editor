# FTP Editor

**Edit codes directly from FTP**

*Not ready yet :(*

## Install
```bash
# Clone repository
git clone git@github.com:tkinali/FTP-Editor.git <folder_name>

# Enter directory
cd <folder_name>

# Install dependencies and start
npm install && npm start
```
